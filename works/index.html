Gene Canvas
Mid 2015
HTML5 Canvas
Data analytics

Gene Canvas는 대용량의 유전자 데이터를 셀 구조로 시각화해주는 웹 어플리케이션입니다.

결과 필터링, 순서 변경등이 가능하며, HTML5 Canvas 2D Context를 이용하여 데이터를 렌더링합니다.

Epidemic Simulator
Early 2020
WebGL
Simulation
Optimization

Epidemic Simulator는 3Blue1Brown의 Simulating an epidemic 영상에 영감을 받아 이틀에 걸쳐 제작한 토이 프로그램입니다.

SIR 모델을 기반으로 여러가지 전염병에 관련된 변수를 조작하여 전염병 확산 과정을 간단하게 시뮬레이션 해볼 수 있습니다.

WebGL 2.0이 사용되었으며, 대량의 데이터를 시뮬레이션하고 렌더링하기 위한 필수적인 최적화들이 포함되었습니다.

Gene Tree
Early 2017
Electron
HTML5 Canvas
Data analytics

Gene Tree는 유전자 데이터를 방사형 이진 트리 구조로 시각화해주는 웹 프로그램입니다.

Electron을 사용하였으며, 데이터 렌더링에는 HTML5 Canvas가 사용되었습니다.

확대/축소, 회전등이 가능하며, 보고있는 화면을 10,000x10,000 해상도 수준의 그림으로 저장할 수 있습니다.

Figure Module
Early 2016
HTML5 Canvas
AngularJS

Figure Module은 PowerPoint에서 사용할 수 있는 차트, 테이블, 도형등을 웹에서 구현한 웹 모듈입니다.

각 요소들은 크기 조절 / 회전등이 가능합니다. 렌더링에는 HTML5 Canvas가 사용되었습니다.

AngularJS가 사용되어 양방향 데이터 바인딩이 가능합니다.

Color Subtitler
Mid 2019
HTML5 Canvas
HTML5 Video
Editor

Color Subtitler는 비디오 영상에 대한 색상 자막 구간을 편집할 수 있는 웹 어플리케이션입니다.

각 구간마다 단일 색상 또는 패턴 색상을 적용할 수 있습니다.

색상 구간은 그룹화가 가능하며, 그룹화된 구간은 프리셋으로 저장하여 사용할 수 있습니다.

복사, 붙여넣기, 실행 취소(Undo), 다시 실행(Redo)등의 기능을 제공합니다.

Olive/Nlive
Mid 2019
Electron
WebGL
React
OffscreenCanvas
AudioWorklet
Node
FFMPEG
Qt

Olive/Nlive는 Premiere Pro 스타일의 영상 편집 프로그램입니다.

개인 취미 프로젝트로 진행하였으며, 현재는 기본적임 타임라인 편집 / 영상 효과 / 영상 재생등의 기능을 제공합니다.

Olive는 Electron 을 기반으로 React / WebGL / OffscreenCanvas / AudioWorklet / FFEMPG 등

최신 웹 기술과 C++ 네이티브 모듈을 사용하여 구현한 버전이며,

Nlive는 같은 내용을 C++ Qt 기반으로 다시 구현한 버전입니다.

Repo https://github.com/orange4glace/nlive
Repo https://github.com/orange4glace/olive
